# oauth2
SpringBoot Web using oauth2

1. web(hello shop)에서 회원가입 후 로그인  -> 관리자 또는 일반 사용자 권한
2. oauth2(google) 연동으로 회원가입 및 로그인  -> 일반 사용자 권한
